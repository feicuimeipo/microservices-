#验证码
1.常规验证码picture 常规的验证码由数字和字母构成，用户输入不区分大小写，可变形成汉字验证。
2.运算验证码compute 运算验证码主要通过给出数字的加减乘运算，填写运算结果进行验证。
3.滑动验证码slide 通过简单的滑动即可完成验证，应用与移动端体验很好。
4.拼图验证码puzzle 拼图。
5.选字验证码pick 通过按顺序点选图中的汉字完成验证，ie浏览器要求9或以上。
